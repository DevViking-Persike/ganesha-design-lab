# ADR-0002 Governança Design System

## Status

Aceito

## Contexto

O DS já existe, mas faltavam regras explícitas de evolução.

## Decisão

Governar componentes por:

- tokens `--gns-`
- convenção `Gns*`
- distinção entre base, composite e showcase
- requisitos mínimos de estados e acessibilidade

## Consequências

- menos inconsistência
- evolução mais previsível dos contratos
