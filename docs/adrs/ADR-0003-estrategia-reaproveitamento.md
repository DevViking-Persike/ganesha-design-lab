# ADR-0003 Estratégia de Reaproveitamento

## Status

Aceito

## Contexto

O projeto já possui ampla base reutilizável e corre risco de duplicação nas páginas-lab.

## Decisão

Antes de criar novo:

- pesquisar `DesignSystem`
- pesquisar `Composites`
- extrair novo composite só com repetição comprovada

## Consequências

- menos wrappers redundantes
- mais coerência entre catálogo e biblioteca
