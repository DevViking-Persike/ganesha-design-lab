# Visão Geral do Frontend

O frontend do Ganesha Design System é um host web em Blazor Server apoiado por uma biblioteca Razor compartilhada que concentra o Design System, os componentes compostos, os serviços globais de UI e o catálogo de demonstração.

Hoje o projeto funciona principalmente como design lab e biblioteca visual. O host web em `src/Ganesha.DesignLab.Web` sobe a aplicação; o núcleo funcional está em `src/Ganesha.DesignLab.Shared`.
