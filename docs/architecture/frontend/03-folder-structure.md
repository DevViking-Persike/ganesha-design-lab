# Estrutura de Pastas

## `src/Ganesha.DesignLab.Web`

- `Program.cs`: DI, pipeline e renderização interativa
- `Components/App.razor`: shell HTML e assets
- `Components/Routes.razor`: router principal
- `Components/Layout`: layout mínimo do host

## `src/Ganesha.DesignLab.Shared`

- `Components/DesignSystem`: componentes base
- `Components/Composites`: composições reutilizáveis
- `Components/Infrastructure`: tema global
- `Components/Lab`: layout e páginas de catálogo
- `Services`: serviços de UI
- `Models`: contratos de suporte
- `wwwroot/css`: tokens, base, temas, utilitários, páginas
- `wwwroot/js`: interatividade complementar
